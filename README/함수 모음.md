```java
strip : 문자열의 양쪽 공백을 모두 지워라 함수

stripTrailing() : 오른쪽 공백 제거
```

```java
replace : 특정 문자를 치환해주는 기능이다. (  치환 : 바꾼다 )

String str = "apple";
System.out.println(str.replace('p', 'b'));

출력 : abble
```

# 문자열 확인(조회) 함수


```java
 length() : 길이 보기
String str = "hello";
System.out.println(str.length());

 결과: 5
```
```java
 charaAt() : 특정 위치 문자 보기 **
String str = "hello";
System.out.println(str.charAt(1));

결과 e      (index 0부터 시작해서 hello중 "e" 나옴)
```
```java
 contains() : 포함 여부 확인
String str = "hello";
System.out.println(str.contains("ell"));

결과 : true
```
```java
indexOf() : 위치 찾기 **
String str = "hello";
System.out.println(str.indexOf("l"));

결과 2      (index 0부터 시작해서 "hello" l은 2번째 위치)
```
